Husqvarna

Au lancement du projet une fenêtre s'affiche dans laquelle il y a :
Les deux tondeuses
Un formulaire
La position des tondeuses

On insère dans les champs les deux séquences souhaitées. Lorsqu'on clique sur "valider",
les tondeuses effectuent leurs actions et leurs positions s'actualisent.

Si on clique sur "valider" alors que les champs sont vide, un message s'affiche dans
la console pour prévenir l'utilisateur, les tondeuses ne se lancent pas.
